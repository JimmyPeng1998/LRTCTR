# LRTCTR

A Matlab solver for **l**ow-**r**ank **t**ensor **c**ompletion in **t**ensor **r**ing decomposition.



## How to run?

1. Run ``install.m`` 
2. Run ``Test_Synthetic_Noiseless.m`` for noiseless observation
3. Run ``Test_Synthetic_Noisy.m`` for noisy observation
